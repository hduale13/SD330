Hodo's Design Principles

Key principles:

  • Visability
  • Consistency
  • Feedback
  • Constraints
  • Affordance
  • Accessibility

Cognitive design principles:

  • Attention
  • Perception
  • Memory

Emotional/Accessibility/Error Principles:

  • Be empathetic to users
  • Make feedback immediate
  • Accessibility can be handled easily:
    • Vision: work on the text or color
    • Sound: User can change the volume, or use captions for videos
  • Accessibility:
    • Be robust
    • Operable
    • Understandable
    • Be Perceivable

Complexity Principles:

  • Use icons
    • Ensure the user can understand them
  • Allow the user to learn by doing it
  • Add your contact info:
    • Email
    • Chat feature
    • Phone

Cultural/Gender principles:

  • Every culture is not the same
    • Some are more masculine vs femenine
    • Some view individualis over communitarism
  • Market products based on your target
    • Using description words or colors

Conceptional principles:

  • Design the conceptional model purposefully
    • Not based solely on technology
  • Be aware of poor or bad models
  • Make the mental and conceptional models match

Human Centered design principles:

  • Users should be able to determine hwat actions are possible
  • Provide feedback to users
  • Ensure you have good mapping
    • The relationship between controls and their actions
